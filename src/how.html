How to Play

Overview

Minesweeper is a single player puzzle video game.
The objective is to clear a rectangular board containing hidden mines without detonating them.
The player is initially presented with a grid of unknown squares.
Some random squares, unknown to the player, contain mines.
The size and difficulty is choosen by the player beforehand.
The game is played by clicking/selecting a square.
If the square contains a mine, the game is lost.
If no mine, a number is displayed of how many adjacent mines are near the square selected.
If no mines are adjacent, the square is blank.
Using this number, the player has to avoid mines to select every last non-mine square.
